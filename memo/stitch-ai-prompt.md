## Stitch AI Prompt

```
設計一個專業的機能食品品牌網站「food+ 食之味」

【品牌定位】
- 由美國註冊營養師創辦的機能食品品牌
- 主打「全食物營養」、「無添加」、「植萃機能」
- 品牌理念：讓健康變簡單，讓生活更有味

【目標受眾】
- 50-75 歲年長者
- 注重健康養生、追求天然食品
- 希望透過飲食改善睡眠品質、延年益壽
- 不喜歡複雜的瓶瓶罐罐保健食品

【視覺風格】
- 色調：暖色系為主（米白、淺橘、暖棕、淡金色）
- 輔助色：自然綠、淺木色，營造溫馨舒適感
- 風格：專業醫療級質感 + 溫暖親切的居家氛圍
- 字體：大字級、高對比、易於閱讀
- 圖片：真實食材、年長者健康生活場景、晨間/睡前儀式感

【網站結構】
1. 首頁 (Home)
   - Hero 區塊：強調「好眠養生，從一杯開始」
   - 核心價值：全食物營養、無添加、植萃助眠
   - 信任背書：營養師專業認證、產品檢驗報告
   - 使用情境：晨間活力、午後補給、睡前安神

2. 關於我們 (About)
   - 創辦人故事（營養師背景）
   - 品牌使命：讓長輩不再依賴保健食品
   - 專業團隊介紹

3. 產品頁面 (Products)
   - 清晰的產品分類
   - 大圖展示、簡潔說明
   - 營養成分表、功效說明
   - 簡單的購買流程

4. 健康專欄 (Blog)
   - 睡眠與營養知識
   - 長者養生秘訣
   - 食譜分享

5. 聯絡我們 (Contact)
   - LINE 官方帳號（一對一諮詢）
   - 大按鈕、簡易表單

【特殊設計需求】
- 大按鈕、大字體，適合年長者操作
- 簡潔導航，減少層級
- 高對比度配色，確保可讀性
- 信任元素：專業認證標章、客戶見證、媒體報導
- 強調關鍵字：好眠、養生、天然、安心、專業營養師

【情感訴求】
- 傳達溫暖、關懷、專業、值得信賴
- 讓年長者感受到「被照顧」而非「被推銷」
- 營造「送給父母最好的禮物」的氛圍

【參考元素】
- 日式簡約養生風格
- 醫療保健網站的專業信任感
- 茶飲品牌的溫潤質感
```
